# SamsungThemeMod
 
---
 
Converts proprietary Samsung Galaxy firmware-embedded theme APKs, such as TB or MM themes, into standard 10-minute trial APK themes.

---
 
## Support Devices (Pls Check each branch)
 
| Theme | UI Version | Phone  | Android | Compatibility           |
| ----- | ---------- | ------ | ------- | ----------------------- |
| MM    | One UI 7.0 | ZFlip4 | 15      | >= Android 9 / One UI 1 |
